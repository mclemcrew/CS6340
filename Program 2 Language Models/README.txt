Programming Language/Version: Python3

Instructions:
To run the n-gram language modeler, execute the following commands:
python3 langmodels.py train1.txt -test test1.txt
OR
python3 langmodels.py train2.txt -test test2.txt

To run the sentence generator, execute the following command:
python3 langmodels.py train2.txt -gen seeds.txt

Tested on computer lab2-11.eng.utah.edu
OS: Red Hat Enterpirse Linux 8.4

Limitations: All the bigrams in the dictionary are lowercase so anything generated by the sentence generator will be lowercase after the initial seed.
